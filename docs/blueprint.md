# **App Name**: TutorGo

## Core Features:

- Tutor Registration: Tutors can register and create profiles with their subjects, hourly rate, demo video, and availability.
- Student Search: Students can search for tutors by location, subject, and hourly rate range using geospatial queries.
- Real-time Chat: Students and tutors can communicate in real-time via one-to-one chat using Socket.IO.
- Booking System: Students can book available slots with tutors, and tutors can accept or reject the bookings.
- Payment Integration: Integrate a mock payment flow using Stripe test mode to track transactions.
- Profile Moderation: AI tool that assists admins in moderating tutor profiles and demo videos.
- Geolocation Services: The Next.js application enables searching for tutors within a specified radius of the student's location using latitude and longitude inputs.

## Style Guidelines:

- Primary color: Deep Blue (#1E3A8A) to evoke trust and competence.
- Background color: Light Gray (#F9FAFB) for a clean, modern look.
- Accent color: Teal (#008080) for CTAs and important UI elements.
- Body font: 'PT Sans', a humanist sans-serif, combining a modern look with warmth for readability.
- Headline font: 'Space Grotesk', a sans-serif, with a computerized, techy, scientific feel. It pairs well with 'PT Sans'.
- Code font: 'Source Code Pro' for displaying code snippets.
- Use clean and professional icons for navigation and key features.
- A clean and intuitive layout with clear sections for search, profiles, booking, and chat.
- Subtle animations and transitions to enhance user experience, such as loading indicators and smooth scrolling.
